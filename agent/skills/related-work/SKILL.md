# Related-Work Skill

Write a short related-work section grounded only in papers already filed in
`/memories/library/`.

## Format
- 1-2 paragraphs, ~150-250 words, narrative (not a bulleted list).
- Group papers by theme; contrast approaches rather than listing them.
- Cite inline with numbered brackets `[1]`, `[2]`, in order of first mention.
- End with a numbered **References** list; each line is `[n] Authors (year), Title. arXiv:id`.

## Rules
- Only cite papers actually in the library — never invent a reference.
- If the library lacks coverage for a claim, say so instead of padding.
